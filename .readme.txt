                조작 설명서
  o/p  : 투영을 선택한다 (직각 투영 / 원근 투영)
  7/8  : 육면체들의 움직이는 속도 증가 / 감소
  9/0  : 육면체들의 최대 높이값 증가 / 감소
  z/Z  : 원근 투영 시 z축으로 이동할 수 있게한다.
  m/M  : 육면체들이 위 아래로 움직인다 / 멈춘다.
  y/Y  : 카메라가 바닥의 y축을 기준으로 양 / 음 방향으로 회전한다.
   v   : 육면체들 움직임이 멈추고 낮은 높이로 변한다. / 원상복귀
   r   : 미로 생성
   s   : 로봇 생성
 1/2/3  : 카메라 시점 1, 3인칭 변환 / 2: 맵뷰(기본카메라)
 방향키: 로봇 조작
  +/-  : 로봇의 속도 증가 / 감소
   c   : 초기화 후 재입력
   q   : 프로그램 종료

직각/원근 투영 선택은 맵뷰상태일때만 가능합니다.
z축 이동은 맵뷰상태일때만, 직각/ 원근 모두 가능합니다.
z축 이동은 오로지 z축만 이동하기 때문에 y로테이션이 적용된 상태에서 물체방향으로 정확히 이동하지 않습니다.
육면체가 나타나는 애니메이션은 바닥에서부터 치솟게 / 미로 블럭은 전체가 파도치는 형상으로 작성하였습니다.
미로 제작은 백트래킹방식으로 작성하였고 맵의 블럭 개수를 고려한 랜덤한 미로가 생성됩니다.
미로는 입구는 고정, 출구는 랜덤입니다.
미로는 블럭 개수가 많을 때 더욱 복잡한 미로가 생성되게 구현했습니다.
v키입력시 블럭의 속도가 서서히 감소해서 멈추게 만들었습니다.
+/-는 로봇의 속도 증가 감소로 바꾸었고 블럭의 속도 증감은 7/8에 구현했습니다
로봇의 크기는 미로 블럭의 사이즈에 비례하게 만들었습니다.
방향키로 로봇을 움직일수있으며 움직일때 애니메이션을 구현하였습니다.
미니맵은 미니맵이라는 특성에 맞게 맵 전체를 보여주는 고정형 직교투영으로 구현했습니다.
모든 값 초기화 시에는 입력값을 다시 넘겨주어야 초기화 됩니다. (콘솔창확인)
